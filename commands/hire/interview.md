Run the Interview Questions exercise from `~/.claude/skills/hiring/SKILL.md` (Stage 3: Interview Questions).

1. Ask what role they're hiring for
2. Ask what competencies they want to assess (or suggest based on role)
3. Generate structured questions organized by category: behavioral, technical, problem solving, culture add, motivation
4. For each question, include what a GOOD answer looks like and what a RED FLAG answer looks like
5. Organize questions by interview round (screen, technical, hiring manager, culture)
6. Include the illegal/avoid reminder — never ask about age, marital status, children, religion, disability, nationality
7. Provide scoring criteria per question

Same questions, same order, same scoring for every candidate. Structure beats intuition.
