Run the Interview Scorecard exercise from `~/.claude/skills/hiring/SKILL.md` (Stage 2: Interview Scorecard).

1. Ask what role they're hiring for
2. Ask what competencies matter most (technical, problem solving, communication, culture add)
3. Generate role-specific competencies (not generic ones)
4. Build the full scorecard with 1-5 scoring for each competency
5. Include the scoring guide (5=exceptional through 1=significant concern)
6. Add recommendation options: Strong Hire / Hire / No Hire / Strong No Hire
7. Include a red flags section

Remind them: fill out scorecards independently before debrief to avoid groupthink.
